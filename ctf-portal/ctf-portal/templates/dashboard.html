Tool guide

nmap / Zenmap

Scans a host for open ports and running services.
nmap -sV {{ target_url.split('//')[-1].split('/')[0].split(':')[0] }}

whois

Looks up domain registration info (owner, registrar, dates).
whois example.com

nslookup / dig

Queries DNS records — IP addresses, mail servers, subdomains.
nslookup example.com

gobuster

Brute-forces hidden directories/files using a wordlist.
gobuster dir -u {{ target_url }} -w wordlist.txt

dirbuster

GUI alternative to gobuster — same idea, point-and-click.
